# Torre_de_Hanoi

esse é um programa que resolve o jogo de raciocínio lógico, Torre de Hanoi, aplicando o recurso da recursão em C++.
Vale dizer que é resolvido independentemente do número de discos, mostrando passo a passo quais discos mover. 
Além disso, conta a quantidade de clocks por segundo e quantas vezes a recursção foi usada, o que facilita a análise de complexidade.
